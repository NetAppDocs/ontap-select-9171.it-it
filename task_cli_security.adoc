= Cambia la password dell'amministratore di Deploy

È possibile modificare la password per l'account amministratore della macchina virtuale Deploy in base alle proprie esigenze, utilizzando l'interfaccia della riga di comando.

.Passi
. Sign in all'interfaccia della riga di comando dell'utilità Deploy utilizzando l'account amministratore.
. Cambia la password:
`password modify`
. Rispondi a tutti i prompt in modo appropriato al tuo ambiente.
